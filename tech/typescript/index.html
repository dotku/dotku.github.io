TypeScript

TypeScript 由微软开发，是一种强数据类型的 JavaScript 衍生语言。

比如以下例子 

function greeter(person: string) {
    return "Hello, " + person;
}

var user = "Jane User";

document.body.innerHTML = greeter(user);

注意在函数声明这一句，function greeter(person: string)，不同于 JavaScript 的 Dynamically Type，TypeScript 通过这种方式来限制数据类型。

所以在这里 Person 必须是 String 类型，否则在编译成 JavaScript 过程中会出错。

仅仅是这样，还不够，TypeScript 还支持 Class 和 Interface 的方式来处理复杂的数据关系。

class Student {
    fullName: string;
    constructor(public firstName, public middleInitial, public lastName) {
        this.fullName = firstName + " " + middleInitial + " " + lastName;
    }
}

interface Person {
    firstName: string;
    lastName: string;
}

function greeter(person : Person) {
    return "Hello, " + person.firstName + " " + person.lastName;
}

var user = new Student("Jane", "M.", "User");

document.body.innerHTML = greeter(user);

虽然 TypeScript 是由微软开发维护的，然而现在连 Google 都已经开始使用 TypeScript 了，可见 TypeScript 意义重大。
