# Less and More Command

When you use `ls` command, it might be too long for the screen. You can sue `less` to pipe the command, such as 'ls /bin | less'. Then you can use up and down key to sroll the page.

Use `more` by default, it doesn't support scroll feature, but later on, starts from 1992.
